Useful Sbt Commands
-------------------

Generate intellij configuration files
```shell
sbt> gen-idea
```

Autocompile and launch tests on file changes
```shell
sbt> ~ ;compile ;test
```

Create jar file for submission (submission\*\*\*.jar)
```shell
sbt> package-for-submission Engels Rutz Pittet
```
